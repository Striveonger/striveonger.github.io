VMware Fusion 配置网卡IP

Mr.Lee 2021-01-27 11:53:00 LinuxVM

今天把系统升到了 MacOS Big Sur 11.1 版本, 结果VirtualBox 不能运行了...升级后, VM也还是跑不起来....决定弃坑, 换到 VMware Fusion .

那么问题了, 装好了系统后, 默认IP是192.168.87.1 原来的IP用熟了, 就感觉这个新的很怪异...

改起来吧

# 修改网卡IP

# 打开 VMware Fusion 的网卡文件
sudo vim /Library/Preferences/VMware\ Fusion/networking
1
VERSION=1,0
answer VNET_1_DHCP yes
answer VNET_1_DHCP_CFG_HASH E296FFB011377971A5054508226C7F8EE89C5A7C
answer VNET_1_HOSTONLY_NETMASK 255.255.255.0
answer VNET_1_HOSTONLY_SUBNET 192.168.135.0
answer VNET_1_VIRTUAL_ADAPTER yes
answer VNET_8_DHCP no
answer VNET_8_DHCP_CFG_HASH 27587C2414FAA714951196A664B669ABBA39F927
answer VNET_8_HOSTONLY_NETMASK 255.255.255.0
answer VNET_8_HOSTONLY_SUBNET 10.13.144.0
answer VNET_8_NAT yes
answer VNET_8_VIRTUAL_ADAPTER yes
add_bridge_mapping en0 2
